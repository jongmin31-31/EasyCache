# EasyCache Overview
**Database > EasyCache > Overview**
NHN Cloud EasyCache is a service that provides Valkey in a cloud environment. You can use a highly available in-memory cache server with Valkey installed with simple settings.

## Main Features
- Easily use the Valkey in-memory cache with the specifications you need.
- High availability features can be configured to ensure availability even in the event of a failure.
- EasyCache can automatically or manually perform backups at a time you want, and you can restore the cache using the backups.
- You can check events occurring in EasyCache and subscribe to receive related information via email or SMS.
- You can manage cache settings as parameter groups to quickly and easily apply them to multiple caches.
- You can quickly and easily configure security rules for cache access.
- A server dashboard provides an at-a-glance overview of cache status.
- The server dashboard allows you to view various metrics for instances and databases in graphs.

!!! tip "Note"
    EasyCache can only be used by enabling Compute, Network service.
